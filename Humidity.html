Humidity

Initial plot of all city's latitudes against Humidity (in %) shows no correlation.

If we do separate plots of cities with latitudes in North and South hemispheres, we can do a linear regression to see if a correlation emerges.



And we see that indeed there is no significant correlation, with the respective R Square values shown. Thus we conclude that Humidity does not vary with Latitude.
